# Coffee Shop - Triple Espresso

Este proyecto es una página web diseñada para una cafetería ficticia llamada **Coffee Shop - Triple Espresso**.  
El objetivo del sitio es presentar información clave como recetas, un formulario de reservación y una sección de contacto, siguiendo la maqueta proporcionada en el brief del Sprint 4.

---

## Tecnologías utilizadas

El proyecto está desarrollado utilizando únicamente las tecnologías básicas solicitadas:

- **HTML5**
- **CSS3**

---

## Descripción del proyecto

El sitio web está compuesto por las siguientes secciones principales:

### **Encabezado**

- Menú de navegación
- Imagen decorativa
- Título y descripción de la cafetería

### **Sección de Recetas**

- Videos integrados desde YouTube mediante iframes
- Subtítulos y diseño organizado

### **Formulario de Reservación**

- Campos para nombre, correo, fecha, hora y número de personas
- Checkbox para aceptar términos de uso
- Botón de envío estilizado

### **Pie de página**

- Círculo decorativo azul
- Logo colocado con medidas exactas
- Sección de redes sociales
- Copyright alineado al diseño del brief

---

## Planes de mejora

En futuras versiones, sería ideal implementar:

- **Diseño responsive**, para adaptar el sitio a móviles y tablets.
- **Animaciones suaves**, como efectos de hover avanzados o transiciones.
- **Validaciones interactivas** para el formulario con JavaScript.
- **Optimización de imágenes** para mejorar el rendimiento.
- **Modo oscuro** o variaciones de tema.
